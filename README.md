# Olympia DAO

## Olympia Upgrade – Ethereum Classic

This repository contains a minimal, static website that explains the Olympia ECIP suite for Ethereum Classic.

**Included pages:**
- `index.html` — High-level introduction to Olympia  
- `upgrade.html` — Overview of ECIPs 1111–1115  
- `fees.html` — Fee mechanics & miner incentives (ECIP-1111 & ECIP-1115)  
- `treasury.html` — Treasury contract overview (ECIP-1112)  
- `governance.html` — Governance & funding process (ECIP-1113 & ECIP-1114)  
- `impact.html` — Ecosystem impact & use cases  
- `roadmap.html` — Roadmap, rollout phases, and milestones  
- `vision.html` — Long-term vision for ETC and Olympia  

**Purpose:**  
Provide an accessible, educational companion to the official ECIP specifications.

**Canonical Specifications:**  
All Olympia ECIPs (1111–1115) live in the official Ethereum Classic ECIP repository:  
https://github.com/ethereumclassic/ECIPs
